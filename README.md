# react-flux-example
Purely for learning purpose, Uses McFly Flux solution


### How to run

Make sure gulp is installed globbaly, if not :
```
npm install gulp -g
```

To install all dependecies :
```
npm install
```

To run the app :

```
gulp serve
```
To execute tests :

```
npm test
```
